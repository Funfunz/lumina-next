# Button component

## Concepts

A Button is a clickable element that can trigger events on a page. 
 
Use Buttons provide interaction capabilities to your component.

## Usage

```jsx
// my-react-script.jsx
<Button
  label="Some label"
  onClick={() => { console.log("Button clicked...") }}
  primary | secondary | large | small
/>
```
