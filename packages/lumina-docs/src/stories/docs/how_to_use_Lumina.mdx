# How to use Lumina

It should contains all steps from from starting an application, either using Next as or vite or any other tool, up to having Lumina running on the screen

## Initial application

### NextJS

[Initial application with NextJS](https://nextjs.org/docs/getting-started/installation)

```sh
npx create-next-app@latest
```

### Vite

[Initial application with Vite](https://vitejs.dev/guide/)

```sh
npm create vite@latest my-vue-app -- --template vue
```

```sh
yarn create vite my-vue-app --template vue
```

```sh
pnpm create vite my-vue-app --template vue
```

```sh
bun create vite my-vue-app --template vue
```

## Installing Lumina

Using npm:

```sh
npm install
```

## Add Lumina to project

### Import Lumina

```ts
import Lumina from '@lumina/core'
import luminaConfig from './luminaComponents/config'
```
